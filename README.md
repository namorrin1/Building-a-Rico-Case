# RICO CASE
Fully Functional Finch 2.0 application designed for play and experimentation. 
Features include:
- Manual Control Mode
  - Allows user to use WASD movement controls with Q as a killswitch and Spacebar as a stop-in-place switch
- Write Mode
  - Allows user to write a message (<15 characters long) straight to the LED array on the Finch's tail
- Roomba Mode
  - Allow user to toggle a "Roomba" mode that has the Finch move forward until it detects an object in front of it, which it then turns left and continues to move forward until a user enters 'STOP' 
- Draw Mode
  - Allows user to choose between a Square, a Hexagon, and a Star to draw below the Finch with a marker. The user can also determine the length of the sides of each shape respectively.
- Song Mode
  - Allows user to choose between three lullabies that the Finch will peacefully sing. The three lullabies consist of Mary Had a Little Lamb, Oh When the Saints Go Marching In, and Twinkle Twinkle Little Star.
